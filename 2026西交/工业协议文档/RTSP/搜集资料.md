# RTSP (Real Time Streaming Protocol) — 搜集资料

## 1. 核心规范文档

### RFC 2326 — RTSP 1.0（原始标准）
- **链接**: https://www.rfc-editor.org/info/rfc2326/
- **发布**: 1998年4月
- **作者**: H. Schulzrinne, A. Rao, R. Lanphier
- **说明**: RTSP 1.0 正式规范，定义流媒体控制协议的基础框架
- **状态**: 已被 RFC 7826 取代

### RFC 7826 — RTSP 2.0（最新标准）
- **链接**: https://datatracker.ietf.org/doc/html/rfc7826
- **发布**: 2016年12月
- **说明**: RTSP 2.0，与 1.0 不完全兼容，新增 PLAY_NOTIFY 等方法，废除 RECORD 等方法

### [MS-RTSP] — Microsoft RTSP 扩展
- **链接**: https://learn.microsoft.com/es-es/openspecs/windows_protocols/ms-rtsp/d9c6f2be-6778-4531-971d-2d295b568d52
- **说明**: Windows Media Services 的 RTSP 实现扩展规范

## 3. 工控场景应用

RTSP 在工控中主要用于:
- 工业视频监控（IP Camera 视频流传输）
- 结合 ONVIF 标准用于摄像头发现和管理
- 工业现场远程可视化监控
