# TRDP (Train Real-time Data Protocol)

## 协议概述

**编号**: 53/59

**描述**: 列车实时数据协议 — IEC 61375 标准，用于列车通信网络（TCN）

| 属性 | 内容 |
|------|------|
| **标准** | IEC 61375-2-3 |
| **端口** | 以太网 |
| **组织** | IEC TC 9 |
| **分类** | 轨道交通 |
| **传输层** | TCP/UDP |

## 典型应用场景

- IEC 61375 标准，用于列车通信网络（TCN）

## 安全风险

1. **认证机制薄弱** — 多数工控协议缺乏强认证
2. **协议实现漏洞** — 厂商实现中的缓冲区溢出等问题
3. **明文传输** — 部分协议无加密
4. **暴露面过大** — 工控协议不应暴露在互联网
