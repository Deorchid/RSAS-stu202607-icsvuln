# HID VertX Edge Controller Protocol

## 协议概述

**编号**: 57/59

**描述**: HID VertX 边缘控制器协议 — 门禁控制系统通信协议

| 属性 | 内容 |
|------|------|
| **标准** | HID 私有协议 |
| **端口** | TCP 可变 |
| **组织** | HID Global |
| **分类** | 门禁安全 |
| **传输层** | TCP/UDP |

## 典型应用场景

- 门禁控制系统通信协议

## 安全风险

1. **认证机制薄弱** — 多数工控协议缺乏强认证
2. **协议实现漏洞** — 厂商实现中的缓冲区溢出等问题
3. **明文传输** — 部分协议无加密
4. **暴露面过大** — 工控协议不应暴露在互联网
