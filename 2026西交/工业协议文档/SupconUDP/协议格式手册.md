# Supcon UDP 协议格式手册

## 1. 协议概述

Supcon UDP 是浙江中控技术股份有限公司（SUPCON）DCS 系统的私有通信协议，用于控制器与 I/O 模块、操作站之间的实时数据交换。

| 属性 | 内容 |
|------|------|
| 协议类型 | 私有 UDP 协议 |
| 用途 | DCS 控制器与 I/O 模块、操作站间的实时通信 |
| 厂商 | 浙江中控技术股份有限公司 |
| 公开资料 | 极少（厂商未公开协议规范） |
| 主要产品 | ECS-700（大规模）、ECS-100（中规模）、Webfield 系列 |

> ⚠️ 本手册内容基于行业通用 DCS 通信模式整理，待补充实际逆向分析数据。

## 2. 已知通信特征

### 2.1 典型 DCS UDP 通信模式

中控 DCS 系统通常将网络分为 **控制网** 和 **系统网** 两层：

```
+---------------+
|  操作站 (HMI) |   系统网（UDP）
+-------+-------+
        |
+-------+-------+
|   控制器      |   控制网（UDP）
+-------+-------+
        |
+-------+-------+
|  I/O 模块     |
+---------------+
```

### 2.2 通信角色

| 角色 | 说明 |
|------|------|
| 控制器 (Controller) | 核心运算单元，周期采集 I/O 数据并响应操作站请求 |
| I/O 模块 | 采集现场信号（模拟量/数字量），通过控制网上报 |
| 操作站 (HMI/Station) | 人机交互界面，从控制器读写数据 |

### 2.3 可能的通信类型

| 类型 | 方向 | 说明 |
|------|------|------|
| 实时数据广播 | 控制器 → 操作站 | 周期上报 AI/AO/DI/DO 数据 |
| 控制指令 | 操作站 → 控制器 | 设定值修改、PID 参数调整 |
| 组态下装 | 工程师站 → 控制器 | 控制策略下载 |
| I/O 采样数据 | I/O 模块 → 控制器 | 周期数据采集 |
| 诊断/报警 | 控制器 → 操作站 | 设备状态、异常报警 |

## 3. 行业通用 DCS 帧结构（参考）

以下结构基于同类 DCS 系统（如和利时、浙大中控早期系统）归纳，Supcon 具体实现可能不同。

### 3.1 通用 UDP 应用层帧

```
+----------------------+
| 帧头标识符 (2-4B)    |  固定魔数，用于帧同步
+----------------------+
| 长度 (2B)            |  总帧长度或数据长度
+----------------------+
| 命令字 (1-2B)        |  功能码（读/写/应答/广播等）
+----------------------+
| 目标地址 (2-4B)      |  站号/控制器地址/模块地址
+----------------------+
| 源地址 (2-4B)        |  发送方地址
+----------------------+
| 数据序号 (2B)        |  包序号，用于去重/重排序
+----------------------+
| 数据体 (可变)        |  实际负载
+----------------------+
| 校验 (2-4B)          |  CRC16/32 或累加和
+----------------------+
```

### 3.2 常见命令类型（推断）

| 命令 | 说明 |
|------|------|
| 0x01 | 读数据请求 |
| 0x02 | 读数据响应 |
| 0x03 | 写数据请求 |
| 0x04 | 写数据响应 |
| 0x10 | 周期数据广播 |
| 0x11 | 报警/事件上报 |
| 0xFF | 心跳/诊断 |

## 4. 建议的协议分析方法

由于 Supcon UDP 是未公开的私有协议，可通过以下途径获取更精确的协议格式：

1. **抓包分析**：在 DCS 系统正常运行时，使用 Wireshark 捕获控制器网段 UDP 数据包
   - 观察数据包周期性规律
   - 通过字段递变识别长度、序号、校验字段
2. **固件逆向**：获取控制器或 I/O 模块固件，分析协议处理代码
3. **驱动文件逆向**：分析上位机软件通信驱动 DLL/SO 文件
4. **厂商技术支持**：联系中控技术部门获取 DCS 系统通信手册
