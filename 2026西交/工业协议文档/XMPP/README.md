# XMPP (Extensible Messaging and Presence Protocol)

## 协议概述

**编号**: 58/59

**描述**: 可扩展消息处理现场协议 — IETF 标准，即时消息/物联网通信（ABB 工控系统使用）

| 属性 | 内容 |
|------|------|
| **标准** | RFC 6120 / RFC 6121 |
| **端口** | TCP 5222 |
| **组织** | IETF / XSF |
| **分类** | 即时通信 |
| **传输层** | TCP/UDP |

## 典型应用场景

- IETF 标准，即时消息/物联网通信（ABB 工控系统使用）

## 安全风险

1. **认证机制薄弱** — 多数工控协议缺乏强认证
2. **协议实现漏洞** — 厂商实现中的缓冲区溢出等问题
3. **明文传输** — 部分协议无加密
4. **暴露面过大** — 工控协议不应暴露在互联网
