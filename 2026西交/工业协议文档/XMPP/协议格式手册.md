# XMPP 协议格式手册

## 1. 核心规范
- **RFC 6120**: https://xmpp.org/rfcs/rfc6120.html
- **RFC 6120 中文**: http://www.rfc2cn.com/rfc6120.html
- **端口**: TCP 5222 (客户端), TCP 5269 (服务器间)

## 2. 协议栈
| 层 | 协议 |
|---|------|
| 消息层 | XMPP Stanza (XML) |
| 安全层 | TLS (STARTTLS) |
| 认证层 | SASL (SCRAM-SHA-1/PLUS) |
| 传输层 | TCP / WebSocket / BOSH |

## 3. Stanza 类型
### <message/> — 异步推送
```xml
<message from='alarm@plc-01' to='operator@scada'>
  <body>ALARM: Temperature exceeds limit</body>
</message>
```

### <presence/> — 状态广播
```xml
<presence from='sensor01@plant' type='available'/>
```

### <iq/> — 请求/响应
```xml
<iq type='get' from='client' to='device' id='1'>
  <query xmlns='urn:xmpp:iot:sensordata'/>
</iq>
```
