# S7COMM 协议格式手册

## 1. 协议栈
| 层 | 协议 |
|---|------|
| 应用层 | S7COMM (0x32) |
| 表示层 | COTP (ISO 8073) |
| 会话层 | TPKT (RFC 1006) |
| 传输层 | TCP 102 |

## 2. TPKT 头 (4字节)
| 偏移 | 大小 | 字段 | 说明 |
|------|------|------|------|
| 0 | 1 | Version | 0x03 |
| 1 | 1 | Reserved | 0x00 |
| 2-3 | 2 | Length | 总长度（含TPKT头） |

## 3. COTP 头
### 连接请求 (CR, 0xE0)
| 偏移 | 大小 | 字段 | 说明 |
|------|------|------|------|
| 0 | 1 | Length | 可变部分长度 |
| 1 | 1 | PDU Type | 0xE0 (CR) |
| 2-3 | 2 | DST-REF | 目标引用，0x0000 |
| 4-5 | 2 | SRC-REF | 源引用 |
| 6 | 1 | Class | 0x00 (Class 0) |
| 7+ | 可变 | Options | TSAP参数 |

### TSAP 编码
| 类型 | 第一字节 | 第二字节 |
|------|---------|---------|
| PG (编程设备) | 0x01 | 机架高4位+槽位低4位 |
| OP (操作面板) | 0x02 | 机架高4位+槽位低4位 |
| S7-1200默认 | 0x01/0x02 | 0x01 (机架0,槽位1) |

## 4. S7COMM PDU 结构
```
[S7COMM头] [参数区] [数据区]
```

### PDU 头 (10字节)
| 偏移 | 大小 | 字段 | 说明 |
|------|------|------|------|
| 0 | 1 | Protocol ID | 0x32 |
| 1 | 1 | ROSCTR | 0x01=Job, 0x02=Ack, 0x03=Ack-Data, 0x07=UserData |
| 2-3 | 2 | Reserved | 0x0000 |
| 4-5 | 2 | PDU Reference | 小端序, 递增编号 |
| 6-7 | 2 | Parameter Length | 参数区长度 |
| 8-9 | 2 | Data Length | 数据区长度 |

### 参数区 — 功能码
| 功能码 | 功能 |
|--------|------|
| 0xF0 | Setup Communication |
| 0x04 | Read Variable |
| 0x05 | Write Variable |
| 0x29 | PLC STOP |

## 5. 地址编码（3字节）
```
Bit 23-21: 未使用 (0)
Bit 20-3:  字节地址 (18位, 范围0-262143)
Bit 2-0:   位地址 (0-7)
```

## 6. 存储区代码
| 代码 | 区域 | 说明 |
|------|------|------|
| 0x81 | I | 输入 |
| 0x82 | Q | 输出 |
| 0x83 | M | 标志位 |
| 0x84 | DB | 数据块 |
| 0x1C | C | 计数器 |
| 0x1D | T | 定时器 |
