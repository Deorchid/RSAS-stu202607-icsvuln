## 3. 技术参数

| 参数 | 值 |
|------|-----|
| 默认端口（未加密） | TCP 1911 |
| 默认端口（TLS 加密） | TCP 4911（FoxS） |
| 认证方式 | Digest / SCRAM-SHA |
| 传输层 | TCP / TLS |
| 通信模式 | 复用式点对点（Multiplexed Peer-to-Peer） |

## 4. 安全建议

- 强制使用 FoxS（禁用未加密的 Fox）
- 禁用 SSLv3，使用 TLSv1.2+
- 替换默认自签名证书为 PKI 证书
- 使用 Strict 模式认证（仅 SCRAM-SHA）
- 不将 Fox/FoxS 端口暴露到互联网
