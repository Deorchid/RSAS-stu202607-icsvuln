# UPnP 协议格式手册

## 1. 协议栈
| 功能 | 协议 | 端口 |
|------|------|------|
| 设备发现 | SSDP (HTTPU/HTTPMU) | UDP 1900 |
| 设备描述 | HTTP | TCP 可变 |
| 控制 | SOAP (HTTP) | TCP 可变 |
| 事件通知 | GENA (HTTP) | TCP 可变 |

## 2. SSDP发现报文
### M-SEARCH 请求
```
M-SEARCH * HTTP/1.1
HOST: 239.255.255.250:1900
MAN: "ssdp:discover"
MX: 3
ST: ssdp:all
```

### 设备响应
```
HTTP/1.1 200 OK
CACHE-CONTROL: max-age=1800
ST: urn:schemas-upnp-org:device:MediaServer:1
USN: uuid:device-UUID::urn:...
LOCATION: http://192.168.1.100:5000/description.xml
```

## 4. 工控风险
- 工业路由器默认WAN口启用UPnP
- 无认证端口映射
- 可被用于OT网络发现和横向移动
