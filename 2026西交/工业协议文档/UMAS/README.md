# UMAS (Unity Modbus Application Protocol)

## 协议概述

**编号**: 55/59

**描述**: 施耐德 Unity Modbus 应用协议 — Schneider PLC 的私有扩展协议

| 属性 | 内容 |
|------|------|
| **标准** | 施耐德私有协议（基于 Modbus） |
| **端口** | TCP 502 (Modbus 端口) |
| **组织** | Schneider Electric |
| **分类** | 工业控制 |
| **传输层** | TCP/UDP |

## 典型应用场景

- Schneider PLC 的私有扩展协议

## 安全风险

1. **认证机制薄弱** — 多数工控协议缺乏强认证
2. **协议实现漏洞** — 厂商实现中的缓冲区溢出等问题
3. **明文传输** — 部分协议无加密
4. **暴露面过大** — 工控协议不应暴露在互联网
