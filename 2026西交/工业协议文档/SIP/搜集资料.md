# SIP (Session Initiation Protocol) — 搜集资料

## 1. 核心规范

### RFC 3261 — SIP 2.0（核心标准）
- **链接**: https://www.ietf.org/rfc/rfc3261
- **发布**: 2002年6月
- **说明**: SIP 2.0 核心规范，定义 SIP 方法、响应码、状态机、URI 方案等
- **状态**: Standards Track（建议标准）

### RFC 3261 全文（RFC Editor）
- **链接**: https://www.rfc-editor.org/info/rfc3261/

## 4. 工控应用

SIP 在工控领域主要用于：
- 工业语音告警系统
- SCADA 系统的音频通信
- 工业对讲系统
- 紧急通信系统
