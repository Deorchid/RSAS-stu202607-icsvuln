# SIP (Session Initiation Protocol)

## 协议概述

**编号**: 50/59

**描述**: 会话发起协议 — IETF 标准，用于 VoIP/视频会议等实时通信，也用于工控语音告警

| 属性 | 内容 |
|------|------|
| **标准** | RFC 3261 |
| **端口** | UDP/TCP 5060, TLS 5061 |
| **组织** | IETF |
| **分类** | 通信协议 |
| **传输层** | TCP/UDP |

## 典型应用场景

- IETF 标准，用于 VoIP/视频会议等实时通信，也用于工控语音告警

## 安全风险

1. **认证机制薄弱** — 多数工控协议缺乏强认证
2. **协议实现漏洞** — 厂商实现中的缓冲区溢出等问题
3. **明文传输** — 部分协议无加密
4. **暴露面过大** — 工控协议不应暴露在互联网
