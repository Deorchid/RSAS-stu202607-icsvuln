# 工控协议检测与分析系统 — 技术文档

## 概述

本系统针对 **工控协议排名 16-20 号** 协议，实现了四大功能模块：
1. **工控协议模拟器** — 模拟真实设备行为
2. **工控协议信息检测逻辑代码** — 协议识别、解析、异常检测
3. **工控漏洞模拟器** — 常见安全漏洞模拟与演示
4. **POC** — 漏洞利用验证代码

## 二、各协议实现状态

### 1. FOCAS (16) — FANUC CNC 机床

| 项目 | 内容 |
|:--|:--|
| **端口** | TCP 8193 |
| **帧结构** | 16字节固定头：长度(2B) + 功能码(2B) + 错误码(2B) + 序列号(4B) + 保留(6B) + 数据 |
| **支持功能码** | 28 个 (读系统信息/轴坐标/动态数据/报警/参数/程序/状态/速度/负载/PMC/宏等) |
| **模拟轴数** | 5 轴 (X/Y/Z + 扩展) |

**检测器**: 解析 16 字节帧头，提取功能码、CNC 型号（特征库匹配 18 种 FANUC 型号），检测 PMC 写入/参数修改/程序修改等高风险操作。提供 `FOCASFlowStats` 流量统计类。

**漏洞**: CVE-2017-16730（未授权访问）、轴数据泄露、参数篡改、程序泄露、PMC 访问、DoS。

### 2. ADS (17) — Beckhoff TwinCAT PLC

| 项目 | 内容 |
|:--|:--|
| **端口** | TCP 48898 |
| **帧结构** | AMS/TCP: `0x0000`(2B) + 总长度(4B) + AMS头(32B) + 负载 |
| **AMS 头** | 目标NetId(6B) + 目标Port(2B) + 源NetId(6B) + 源Port(2B) + 命令ID(2B) + 状态标志(2B) + 数据长度(4B) + 错误码(4B) + 调用ID(4B) |
| **命令** | 1(设备信息), 2(读取), 3(写入), 4(读状态), 5(写控制), 9(读写) |

**检测器**: 精确解析 AMS/TCP 组帧，提取 NetId、端口、命令、错误码、索引组/偏移，检测 WriteControl/AddNotification/大批量写操作等异常。提供 `ADSFlowStats` 流量统计类。

**漏洞**: 未授权访问、变量任意读写、WriteControl 状态篡改、DoS。

### 3. GOOSE (18) — IEC 61850 变电站事件

| 项目 | 内容 |
|:--|:--|
| **EtherType** | 0x88B8 |
| **多播 MAC** | 01:0C:CD:01:00:01 (完整前缀含第4字节) |
| **编码** | ASN.1/BER (Context-Specific tags) |
| **PDU 字段** | gocbRef, timeAllowedtoLive, datSet, goID, t, stNum, sqNum, test, confRev, ndsCom, numDatSetEntries, allData |

**检测器**: BER 解析器提取 GOOSE 所有字段，可检测 stNum=0(伪造)、test=True(测试报文)、ttl=0(丢弃)、非标准 MAC 等异常。

**漏洞**: 报文伪造、重放、stNum 篡改、GOOSE 风暴、Test 标志滥用。

### 4. SV (19) — IEC 61850-9-2 采样值

| 项目 | 内容 |
|:--|:--|
| **EtherType** | 0x88BA |
| **多播 MAC** | 01:0C:CD:04:00:01 |
| **采样率** | 4000 Hz (80采样/周波@50Hz) |
| **通道** | 8 通道浮点: Ua Ub Uc Un Ia Ib Ic In |
| **故障类型** | 三相短路、单相接地、相间故障、过电流、低电压、过电压、零序故障 |

**模拟器**: 合并单元(MU)模拟，7 种故障类型，支持谐波叠加 (3次/5次)、smpSynch 状态设置、自动发包。

**检测器**: BER 解析 SAV PDU + ASDU，提取 svID、smpCnt、confRev、smpSynch、8通道采样值，检测异常电压/电流/同步状态。

**漏洞**: 采样值伪造、篡改、同步攻击、DoS。

### 5. TRDP (20) — IEC 61375 列车控制

| 项目 | 内容 |
|:--|:--|
| **端口** | UDP 17224 (PD), 17225 (MD) |
| **帧头** | 40 字节固定头 (版本/操作码/ComId/ETB ID/源ID/目标ID/序列号/时间戳/TTL/标志位) |
| **信号集** | 21 种 (Speed, BrakePressure, DoorStatus, MotorCurrent, EmergencyBrake, 等) |

**检测器**: 完整 40 字节头解析，提取所有字段含标志位分解，21 种信号类型识别，12 种异常检测规则（紧急制动/跳号/重放/TTL/广播风暴/信号值合理性等）。提供 `TRDPFlowStats` 流量统计与状态追踪类。

**漏洞**: PD 数据伪造、紧急制动注入、车门控制注入、重放攻击、源 ID 欺骗、DoS。

## 四、使用指南

### main.py 入口

```bash
# 列出所有协议
python main.py --list

# 查看协议信息
python main.py --protocol 17

# 运行模拟器 (按 Ctrl+C 停止)
python main.py --run 17 sim

# 运行检测器自测试
python main.py --run 16 detect

# 运行漏洞模拟器
python main.py --run 20 vuln --vuln pd_spoof

# 运行 POC
python main.py --run 18 poc --vuln all

# 运行所有协议的检测器
python main.py --run all detect
```

### 启动模拟器

```bash
# ADS 模拟服务器 (TCP 48898)
python protocols/17-ADS/simulator.py

# FOCAS CNC 模拟 (TCP 8193)
python protocols/16-FOCAS/simulator.py

# GOOSE 变电站事件 (交互式)
python protocols/18-GOOSE/simulator.py
# 命令: trip, close, packet, status

# SV 采样值 (交互式)
python protocols/19-SV/simulator.py
# 命令: fault three_phase, clear, synch, harm, auto, packet, status

# TRDP 列车控制 (交互式)
python protocols/20-TRDP/simulator.py
# 命令: speed 80, brake, release, doors, packet
```

### 运行 POC

```bash
# ADS 未授权访问
python protocols/17-ADS/poc.py --target 127.0.0.1 --vuln unauthorized

# ADS 变量读取
python protocols/17-ADS/poc.py --target 127.0.0.1 --vuln read --offset 0x1000

# FOCAS 未授权访问
python protocols/16-FOCAS/poc.py --target 127.0.0.1 --vuln unauthorized

# FOCAS 全扫描 (5项)
python protocols/16-FOCAS/poc.py --target 127.0.0.1 --vuln all

# GOOSE 报文伪造
python protocols/18-GOOSE/poc.py --vuln spoof

# GOOSE 全扫描 (5项)
python protocols/18-GOOSE/poc.py --vuln all

# SV 采样值伪造
python protocols/19-SV/poc.py --vuln spoof

# SV 全扫描 (4项)
python protocols/19-SV/poc.py --vuln all

# TRDP 紧急制动注入
python protocols/20-TRDP/poc.py --vuln brake_inject

# TRDP 全扫描 (5项)
python protocols/20-TRDP/poc.py --vuln all

# 指定目标发送 (所有 POC 均支持)
python protocols/20-TRDP/poc.py --target 192.168.1.100 --vuln pd_spoof
```

### 运行漏洞扫描

```bash
python protocols/16-FOCAS/vuln_simulator.py --target 127.0.0.1
python protocols/17-ADS/vuln_simulator.py --target 127.0.0.1
python protocols/18-GOOSE/vuln_simulator.py
python protocols/19-SV/vuln_simulator.py
python protocols/20-TRDP/vuln_simulator.py

# 指定单个漏洞类型
python protocols/20-TRDP/vuln_simulator.py --vuln brake_inject
python protocols/16-FOCAS/vuln_simulator.py --vuln list  # 漏洞清单
```
