# UPnP (Universal Plug and Play)

## 协议概述

**编号**: 56/59

**描述**: 通用即插即用协议 — 设备自动发现与互联，在工控中用于网络摄像头和IoT终端

| 属性 | 内容 |
|------|------|
| **标准** | UPnP Device Architecture |
| **端口** | UDP 1900 (SSDP), TCP 可变 |
| **组织** | UPnP Forum (现已移交 OCF) |
| **分类** | 通用协议 |
| **传输层** | TCP/UDP |

## 典型应用场景

- 设备自动发现与互联，在工控中用于网络摄像头和IoT终端

## 安全风险

1. **认证机制薄弱** — 多数工控协议缺乏强认证
2. **协议实现漏洞** — 厂商实现中的缓冲区溢出等问题
3. **明文传输** — 部分协议无加密
4. **暴露面过大** — 工控协议不应暴露在互联网
