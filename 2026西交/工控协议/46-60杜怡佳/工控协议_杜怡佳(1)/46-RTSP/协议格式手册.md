# RTSP 协议格式手册

## 1. 协议概述
- **标准**: RFC 2326 (1.0, 1998) / RFC 7826 (2.0, 2016)
- **传输层**: TCP/UDP，默认端口 554
- **编码**: ISO 10646 (UTF-8)，纯文本协议（类HTTP）
- **官方规范PDF**: https://www.rfc-editor.org/rfc/rfc2326.txt
- **RFC 2326 PDF下载**: https://getrfc.com/rfc2326

## 2. 报文格式

### 请求报文
```
方法 SP 请求URI SP RTSP版本 CRLF
*(通用头 | 请求头 | 实体头)
CRLF
[消息体]
```

### 响应报文
```
RTSP版本 SP 状态码 SP 原因短语 CRLF
*(通用头 | 响应头 | 实体头)
CRLF
[消息体]
```

### 内嵌二进制数据（TCP交错模式）
```
$ <1字节通道ID> <2字节长度(网络序)> <数据>
```

## 3. 方法定义
| 方法 | 方向 | 功能 |
|------|------|------|
| DESCRIBE | C→S | 获取媒体描述 |
| SETUP | C→S | 建立传输通道 |
| PLAY | C→S | 开始播放 |
| PAUSE | C→S | 暂停 |
| TEARDOWN | C→S | 终止会话 |
| OPTIONS | 双向 | 查询可用方法 |
| GET_PARAMETER | 双向 | 获取参数 |
| SET_PARAMETER | 双向 | 设置参数 |

## 4. 状态机
```
Init → (SETUP) → Ready → (PLAY) → Playing
                  Ready → (PAUSE) → Playing
                        → (TEARDOWN) → 结束
```
