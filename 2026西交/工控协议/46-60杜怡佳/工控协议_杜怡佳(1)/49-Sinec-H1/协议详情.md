# Sinec H1 (Siemens Industrial Ethernet) — 协议详情

> **格式参考**: OSI 协议栈分层、帧结构等细节见 [协议格式手册.md](协议格式手册.md)

## 协议演进

Sinec H1 是西门子早期的工业以太网协议，诞生于 1980 年代，用于连接 SINUMERIK 数控系统和 SIMATIC PLC。它是 PROFINET 的前身，目前已基本被取代，但在老厂改造中仍有存量。

## 协议栈

基于 OSI 七层模型实现，**不使用 IP 层**（直接跑在 IEEE 802.3 MAC 层之上），传输层采用 ISO 8073 Class 4 协议。

## 协议特征
- **标准**: 西门子私有协议（基于 ISO 8073 Class 4）
- **物理层**: IEEE 802.3 10Mbps 同轴电缆（后期支持双绞线）
- **无 IP 层**: 早期工业以太网的典型特征，直接在 MAC 层传输
- **应用层**: SINEC AP 1.0 / SICOS / TF（Technology Functions）

## 工控应用场景
- SINUMERIK 840 系列 CNC 控制器间的通信
- SIMATIC S5 系列 PLC 的工厂级联网
- 老厂遗留系统（Brownfield），与 PROFINET 共存

## 安全现状
- 协议设计年代久远，无内置安全机制（无认证、无加密）
- 因无 IP 层，攻击者无法通过标准 IP 路由直接访问，物理隔离是主要防护手段
- 老旧系统通常无法打补丁，风险持续存在

## 相关标准
- ISO 8073 Class 4 (传输层)
- IEEE 802.3 / IEEE 802.2 LLC Class 1
- Siemens SINUMERIK 840 手册
