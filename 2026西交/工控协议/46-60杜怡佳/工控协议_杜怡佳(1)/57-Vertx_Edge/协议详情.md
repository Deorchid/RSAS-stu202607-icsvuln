# HID VertX Edge Controller Protocol — 协议详情

> **格式参考**: 协议端口信息、漏洞利用方式、IDS 检测规则等细节见 [协议格式手册.md](协议格式手册.md)

## 协议演进

discoveryd 是 HID Global VertX Edge 系列门禁控制器的私有通信守护进程，运行在 Linux 嵌入式系统上，监听 UDP 4070 端口。

## 协议栈

基于 UDP 4070 的简单私有协议，无加密、无认证（漏洞版本）。

## 协议特征
- **标准**: HID Global 私有协议
- **端口**: UDP 4070 (discoveryd 守护进程)
- **平台**: 嵌入式 Linux
- **用途**: 门禁控制器发现与管理

## 工控应用 — 物理安全系统
HID VertX Edge 广泛应用于：
- 企业门禁控制系统
- 数据中心机房门禁
- 关键基础设施物理访问控制 (PACS)

## 核心漏洞 (VertXploit)

2016 年，Trend Micro DVLabs 的 Ricky Lawshae 发现 discoveryd 存在 **CVSS 10.0** 未授权命令注入漏洞：
- **根因**: discoveryd 未对网络输入做任何过滤，直接将用户输入拼接到 shell 命令中
- **利用**: 在 `blink` 命令参数中嵌入反引号命令（如 `` `reboot` ``）
- **权限**: 以 root 权限执行任意命令
- **后果**: 远程开锁/关锁、下载持卡人数据库、获取 root shell

该漏洞是 OT 物理安全系统网络风险的典型代表——数字攻击可产生物理后果（非法开门）。
