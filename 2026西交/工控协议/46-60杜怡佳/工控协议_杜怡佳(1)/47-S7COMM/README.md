# S7COMM (Siemens S7 Communication)

## 协议概述

**编号**: 47/59

**描述**: 西门子 S7 PLC 通信协议 — S7-200/300/400/1200/1500 系列 PLC 的工业控制协议

| 属性 | 内容 |
|------|------|
| **标准** | 西门子私有协议 |
| **端口** | TCP 102 |
| **组织** | Siemens |
| **分类** | 工业控制 |
| **传输层** | TCP/UDP |

## 典型应用场景

- S7-200/300/400/1200/1500 系列 PLC 的工业控制协议

## 安全风险

1. **认证机制薄弱** — 多数工控协议缺乏强认证
2. **协议实现漏洞** — 厂商实现中的缓冲区溢出等问题
3. **明文传输** — 部分协议无加密
4. **暴露面过大** — 工控协议不应暴露在互联网
