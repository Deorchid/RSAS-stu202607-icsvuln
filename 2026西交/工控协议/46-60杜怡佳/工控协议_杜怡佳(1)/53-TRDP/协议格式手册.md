# TRDP (Train Real-time Data Protocol) 协议格式手册

## 1. 标准
- **IEC 61375-2-3** — 列车通信网络通信规范
- **协议类型**: 以太网实时数据协议

## 2. 端口
| 数据类型 | 协议 | 端口 |
|---------|------|------|
| Process Data | UDP | 17224 |
| Message Data | TCP | 17225-17226 |

## 3. 协议特征
| 特征 | 说明 |
|------|------|
| 通信模式 | 发布者/订阅者 (PD) + 请求/响应 (MD) |
| 实时性 | 硬实时（微秒级） |
| 安全 | 无加密、无认证、无完整性校验 |
| 开源 | TCNOpen 项目 |

## 4. 安全缺陷
1. 明文传输（无加密）
2. 无认证（任意设备可伪造）
3. 无完整性校验
4. UDP可泛洪
