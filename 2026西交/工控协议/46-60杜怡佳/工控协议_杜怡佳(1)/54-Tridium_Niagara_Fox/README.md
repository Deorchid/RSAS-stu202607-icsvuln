# Tridium Niagara Fox Protocol

## 协议概述

**编号**: 54/59

**描述**: Tridium/Honeywell Niagara 框架 Fox 协议 — 楼宇自控/物联网边缘网关通信

| 属性 | 内容 |
|------|------|
| **标准** | Tridium 私有协议 |
| **端口** | TCP 1911 |
| **组织** | Tridium (Honeywell) |
| **分类** | 楼宇自控 |
| **传输层** | TCP/UDP |

## 典型应用场景

- 楼宇自控/物联网边缘网关通信

## 安全风险

1. **认证机制薄弱** — 多数工控协议缺乏强认证
2. **协议实现漏洞** — 厂商实现中的缓冲区溢出等问题
3. **明文传输** — 部分协议无加密
4. **暴露面过大** — 工控协议不应暴露在互联网
