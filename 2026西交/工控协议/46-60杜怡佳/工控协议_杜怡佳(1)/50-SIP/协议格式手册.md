# SIP 协议格式手册

## 1. 核心规范
- **RFC 3261**: https://www.ietf.org/rfc/rfc3261 （SIP 2.0，2002年6月）
- **端口**: 5060 (UDP/TCP), 5061 (TLS)
- **编码**: 纯文本，类HTTP格式

## 2. 请求报文格式
```
方法 SP 请求URI SP SIP版本 CRLF
*(通用头 | 请求头 | 实体头)
CRLF
[消息体]
```

## 3. 响应报文格式
```
SIP版本 SP 状态码 SP 原因短语 CRLF
*(通用头 | 响应头 | 实体头)
CRLF
[消息体]
```

## 4. SIP方法
| 方法 | 功能 |
|------|------|
| INVITE | 发起会话（三次握手: INVITE→200 OK→ACK） |
| ACK | 确认最终响应 |
| BYE | 终止会话 |
| CANCEL | 取消未完成的请求 |
| REGISTER | 注册用户地址 |
| OPTIONS | 查询能力 |

## 5. SIP响应码
| 分类 | 含义 | 示例 |
|------|------|------|
| 1xx | 临时响应 | 100 Trying, 180 Ringing |
| 2xx | 成功 | 200 OK |
| 3xx | 重定向 | 302 Moved Temporarily |
| 4xx | 客户端错误 | 401 Unauthorized, 404 Not Found |
| 5xx | 服务端错误 | 500 Server Error |
| 6xx | 全局错误 | 600 Busy Everywhere |
