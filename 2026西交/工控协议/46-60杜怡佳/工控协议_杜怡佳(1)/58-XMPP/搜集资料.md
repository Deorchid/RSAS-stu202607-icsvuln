# XMPP (Extensible Messaging and Presence Protocol) — 搜集资料

## 1. 核心规范

### RFC 6120 — XMPP Core（核心标准）
- **链接**: https://xmpp.org/rfcs/rfc6120.html
- **发布**: 2011年3月
- **说明**: XMPP 核心协议规范：XML Streams、STARTTLS、SASL 认证、Stanza 类型
- **中译**: http://www.rfc2cn.com/rfc6120.html

### RFC 6120 原文（RFC Editor）
- **链接**: https://www.rfc-editor.org/info/rfc6120/

## 2. IoT 扩展协议（XEP）

### XMPP IoT 扩展标准页
- **链接**: https://xmpp.org/uses/iot/bindings/
- **说明**: XMPP 在物联网领域的扩展协议索引

### XEP-0323 — Internet of Things - Sensor Data
- **链接**: https://xmpp.org/extensions/xep-0323.xml
- **说明**: 传感器数据通信标准：数据读取、结构化数据、历史数据

### XEP-0325 — Internet of Things - Actuators
- **说明**: 执行器控制标准

### XEP-0326 — Internet of Things - Concentrators
- **说明**: 集中器设备管理标准（管理多个传感器）

### XEP-0347 — Internet of Things Discovery
- **链接**: https://xmpp.org/extensions/xep-0347.pdf
- **说明**: IoT 设备发现（mDNS/DNS-SD、DHCP）

### XEP-0322 — Efficient XML Interchange (EXI)
- **说明**: 高效 XML 交换，适用于 6LoWPAN 等资源受限网络

## 4. 工控应用

### ABB Ability™ 系统中的 XMPP 应用
- ABB 800xA 分布式控制系统使用 XMPP 作为系统间通信协议
- 用于分布式控制中心的实时消息交换
- 告警通知、状态更新、系统间协调

## 5. 技术参数

| 参数 | 值 |
|------|-----|
| 客户端端口 | TCP 5222 |
| 服务器端口 | TCP 5269 |
| 传输层 | TCP, TLS (STARTTLS) |
| 认证 | SASL (SCRAM-SHA-1/PLUS, EXTERNAL) |
| 编码 | XML (可用 EXI 压缩) |
| 通信模式 | 异步消息(Push) / 请求-响应(IQ) / 状态广播(Presence) |
