# S7-PLUS (S7CommPlus) 协议格式手册

## 1. 协议标识
- **Protocol ID**: 0x72（与经典S7comm的0x32区分）
- **传输层**: TCP 102 (TPKT/COTP)
- **加密**: ECC + AES-CTR + HMAC-SHA-256 (V3)
- **TLS模式**: TIA Portal V17+, FW 2.9+（可选用TLS 1.2/1.3）

## 2. 版本对比
| 版本 | 用途 | 安全机制 |
|------|------|---------|
| V1 | S7-1200早期固件 | 单字节反重放 |
| V2 | TIA ≤V12, S7-1500 FW≤1.5 | HMAC-SHA-256完整性 |
| V3 | TIA V13+, S7-1500 FW≥1.8 | ECC密钥交换+AES-CTR+32字节HMAC |

## 3. V3 握手流程
```
1. TCP三次握手 (端口102)
2. COTP CR/CC
3. PLC发送20字节随机Challenge
4. 客户端计算 StructSecurityKey + SessionKey
5. 客户端发送加密密钥材料(180字节)
6. PLC用私钥解密验证
7. 后续所有包带32字节 IntegrityPart
```

## 4. 数据包结构
```
[TPKT] [COTP] [S7-PLUS Header] [IntegrityPart] [EncryptedPayload]
```
