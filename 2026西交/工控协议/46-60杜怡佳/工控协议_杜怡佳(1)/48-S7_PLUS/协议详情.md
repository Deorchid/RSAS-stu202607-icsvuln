# S7-PLUS (Siemens S7-1200/1500 加密通信协议) — 协议详情

> **格式参考**: 版本对比、握手流程、数据包结构等细节见 [协议格式手册.md](协议格式手册.md)

## 协议演进

S7-PLUS（也称 S7CommPlus）是西门子为 S7-1200/1500 系列 PLC 设计的新一代通信协议，Protocol ID 为 0x72（区别于经典 S7comm 的 0x32）。其核心演进路径：

| 版本 | 对应环境 | 安全强度 |
|------|---------|---------|
| V1 | S7-1200 早期固件 | 弱（单字节反重放） |
| V2 | TIA Portal ≤V12 | 中等（HMAC-SHA-256 完整性） |
| V3 | TIA Portal V13+ | 强（ECC + AES-CTR + HMAC） |

V3 引入了完整的加密握手流程，使用 ECC 密钥交换和 AES-CTR 加密，是工控协议中安全设计最先进的之一。

## 协议栈

与经典 S7comm 共用 TCP 102 端口和 TPKT/COTP 传输框架，但在应用层引入了加密载荷。

## 安全机制

V3 版本的安全设计要点：
- **反重放**: Challenge-Response 机制（PLC 发送 20 字节随机数）
- **密钥交换**: ECC (Elliptic Curve Cryptography)
- **加密**: AES-CTR 模式
- **完整性**: 32 字节 HMAC-SHA-256，每个数据包均携带

## 安全研究里程碑
- **DEF CON 25 (2017)**: 首次公开分析 S7CommPlus V3 加密机制
- **IEEE S&P (2023)**: 发现密码实现缺陷，可解密通信

## 相关标准
- 西门子私有协议（V3 起加密保护）
- DEF CON 25 / IEEE S&P 2023 学术研究
