# SV (Sampled Value) — IEC 61850-9-2 — 协议详情

> **格式参考**: 以太网帧字段定义、APDU 结构 (ASN.1 BER)、关键参数等细节见 [协议格式手册.md](协议格式手册.md)

## 协议演进

SV（Sampled Value）是 IEC 61850 标准体系中用于数字化变电站过程层通信的协议。

| 版本 | 说明 |
|------|------|
| IEC 61850-9-2 (2004/2011) | 完整版 SV 规范 |
| IEC 61850-9-2 LE | 轻量版实现指南（行业应用最广） |
| IEC 61869-9 (2016) | 取代 9-2LE 的采样值新标准 |

## 协议栈

SV 直接在以太网链路层传输（Ethertype 0x88BA），上层 APDU 使用 ASN.1 BER 编码，不经过 TCP/IP 协议栈，以保证微秒级实时性。

### 核心特性
- **发布/订阅模式**: MU (合并单元) 发布采样值，IED 保护/测控装置订阅
- **实时性**: 80 样本/周期（保护用，4kHz @ 50Hz）或 256 样本/周期（计量用）
- **时间同步**: 依赖 IEEE 1588 (PTP) 精确到微秒级
- **多播通信**: 使用 01-0C-CD-04-xx-xx 范围的多播 MAC 地址
- **ASDU 合并**: 单个 APDU 可携带 1-8 个 ASDU，减少网络开销

### 工控应用 — 数字化变电站
SV 是智能变电站"三层两网"架构中**过程层网络**的关键协议：
- **MU (合并单元)**: 采集 CT/PT 模拟信号，数字化为 SV 报文
- **IED (智能电子设备)**: 订阅 SV 获取电流/电压采样值，执行保护算法
- **交换机**: 需支持 IEC 61850 多播过滤和 VLAN 隔离

### 安全问题
1. **无认证/加密**: SV 报文不含任何安全字段，接入即可伪造采样值
2. **时间同步攻击**: 破坏 PTP 时钟源可导致采样失步，触发保护误动
3. **多播泛洪**: 攻击者可伪造高采样率 SV 报文，消耗交换机带宽
4. **物理隔离依赖**: 过程层网络通常独立组网，一旦突破边界几乎无防护

## 相关标准

- IEC 61869-9 (新一代采样值标准)
- IEEE 1588 (PTP 精确时间同步)
- IEC 61850-8-1 (GOOSE，同属过程层协议)
