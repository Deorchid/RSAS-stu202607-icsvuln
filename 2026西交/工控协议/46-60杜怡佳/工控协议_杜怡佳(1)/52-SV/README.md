# SV (Sampled Value) — IEC 61850-9-2

## 协议概述

**编号**: 52/59

**描述**: IEC 61850-9-2 采样值协议 — 智能变电站过程层通信，传输电流/电压采样值

| 属性 | 内容 |
|------|------|
| **标准** | IEC 61850-9-2 |
| **端口** | 以太网类型 0x88BA |
| **组织** | IEC TC 57 |
| **分类** | 电力自动化 |
| **传输层** | TCP/UDP |

## 典型应用场景

- 智能变电站过程层通信，传输电流/电压采样值

## 安全风险

1. **认证机制薄弱** — 多数工控协议缺乏强认证
2. **协议实现漏洞** — 厂商实现中的缓冲区溢出等问题
3. **明文传输** — 部分协议无加密
4. **暴露面过大** — 工控协议不应暴露在互联网
