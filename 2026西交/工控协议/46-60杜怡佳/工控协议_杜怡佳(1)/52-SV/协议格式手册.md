# SV (Sampled Value) IEC 61850-9-2 协议格式手册

## 1. 以太网帧结构
| 字段 | 长度 | 说明 |
|------|------|------|
| MAC目的地址 | 6B | 01-0C-CD-04-00-00 ~ 01-0C-CD-04-01-FF |
| MAC源地址 | 6B | 发送设备MAC |
| TPID | 2B | 0x8100 (802.1Q VLAN) |
| TCI | 2B | 优先级信息 |
| Ethertype | 2B | **0x88BA** (SV标识) |
| APPID | 2B | 0x4000 ~ 0x7FFF |
| Length | 2B | 长度 |
| Reserved1 | 2B | 0x0000 |
| Reserved2 | 2B | 0x0000 |
| APDU | 可变 | ASN.1编码采样数据 |
| CRC | 4B | 校验 |

## 2. APDU结构 (ASN.1 BER编码)
```
savPdu (tag 0x60)
├── noASDU (tag 0x80) — ASDU数量
├── Sequence of ASDU (tag 0xA2)
│   ├── svID (tag 0x80) — 采样标识字符串
│   ├── smpCnt (tag 0x82, 2B) — 采样计数器
│   ├── confRev (tag 0x83, 4B) — 配置版本
│   ├── smpSynch (tag 0x85, 1B) — 同步状态
│   └── Sequence of Data (tag 0x87)
│       ├── 通道1: 瞬时值(4B) + 品质(4B)
│       ├── 通道2: ...
│       └── 通道N: ...
```

## 3. 关键参数
| 参数 | 值 |
|------|-----|
| 以太网类型 | 0x88BA |
| APPID范围 | 0x4000 ~ 0x7FFF |
| 采样率 | 80样本/周期 (保护) / 256样本/周期 (计量) |
| ASDU合并 | 1~8个ASDU/APDU |
| 时间同步 | IEEE 1588 (PTP) |
