# UMAS (Unity Modbus Application Protocol) 协议格式手册

## 1. 协议基础
- **传输层**: Modbus TCP (端口502)
- **Modbus功能码**: 0x5A (90) — 用于封装UMAS命令
- **厂商**: Schneider Electric
- **用途**: 配置、监控、编程 Modicon Unity 系列 PLC
- **性质**: 私有协议，无官方公开规范

## 2. 认证机制
| 固件版本 | 认证方式 | 安全性 |
|---------|---------|--------|
| <2.7 | 无/固定Key 0x01 | ❌ 极弱 |
| 2.7~3.5 | 1字节 Session Key (0-255) | ❌ 可暴力破解 |
| ≥3.5 (M340) / ≥4.02 (M580) | Application Password | 🟡 有漏洞 |
